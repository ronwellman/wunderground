# wunderground
Currently displays the most recent weather observation using the Weather Underground API

### Setup
** Not really needed but here just in case **
* `python3 -m venv venv`
* `source ./venv/bin/activate`
* `pip3 install requests`
* `python3 wunderground.py`
* `deactivate`
